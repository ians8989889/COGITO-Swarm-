### 2. OpenClaw Ollama 整合問題解決

**問題：** OpenClaw 無法自動使用 Ollama 模型
**根本原因：** Ollama provider auto-discovery 需要環境變數 + plugin 正確配置

**解決方案：**
1. 設置 `OLLAMA_API_KEY="ollama-local"`（launchctl）
2. 設置 `OLLAMA_HOST="http://127.0.0.1:11434"`（launchctl）
3. 移除 plugin 的 explicit `config` entry（讓 auto-discovery 生效）
4. 重啟 gateway

**驗證結果：**
✅ `openclaw models list --provider ollama` 可見所有模型
✅ Session 已自動切換到 `ollama/qwen3:14b-opt`
✅ Fallback 順序正確：ollama → ollama → claude
