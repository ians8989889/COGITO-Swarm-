# LOCAL PERMISSIONS - 本機讀寫權限 ✅

**生效日期：** 2026-04-02 15:35 GMT+8

## 開放的權限

### ✅ 直接做，無需詢問

- **讀取所有檔案** — workspace 內所有檔案可自由讀取
- **寫入所有檔案** — 新增、編輯、更新檔案
- **執行 shell 命令** — 本機指令、腳本執行
- **備份與版本控制** — commit、push、備份檔案
- **安裝與更新工具** — skills、npm 套件、CLI 工具
- **檔案整理** — 建立目錄、重新命名、移動檔案

### ⚠️ 需要詢問後同意

- **刪除檔案** — `rm`、`trash`、清理檔案（需確認）
- **清空目錄** — 刪除多個檔案或整個資料夾

### ⚠️ 仍需詢問（外部操作）

- **寄信、發文、推文** — 外部通訊
- **API 呼叫** — 需要認證或金費的外部 API
- **上傳、下載** — 外部網路存取（除非是正常工作流的一部分）
- **不確定的操作** — 有疑慮時仍然詢問

## 目的

加快工作流程，減少不必要的對話中斷。本機讀寫都是可追蹤、可恢復的。刪檔需確認是為了防止誤刪。
